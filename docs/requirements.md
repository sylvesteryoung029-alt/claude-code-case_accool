# 需求规格说明书

## 项目名称

房间空调制冷量匹配工具（AC Cooling Calculator）

## 软件概述

开发一款运行于 Windows 设备上的桌面工具，根据用户输入的房间属性，智能匹配所需空调制冷量，并以**匹（HP）**和**千瓦（kW）**两种单位输出结果。软件设计需与 Windows 11 设计语言保持一致，整体以淡蓝色为主色调，并采用透明/毛玻璃视觉效果。

## 目标用户

普通消费者（非专业人士），用于选购家用空调时参考。

## 功能需求

### FR-01 输入参数

用户需在首页填写以下参数：

| 编号 | 参数 | 输入方式 | 备注 |
|------|------|----------|------|
| FR-01.1 | 城市名称 | 下拉选择 | 按气候分区预设城市列表 |
| FR-01.2 | 房间面积 | 数字输入 | 单位：平方米（m²） |
| FR-01.3 | 窗户面积 | 数字输入 | 单位：平方米（m²） |
| FR-01.4 | 所在楼层数 | 数字输入 | 用于判断是否为顶层 |
| FR-01.5 | 是否为顶层 | 复选框 | 仅顶层影响计算 |
| FR-01.6 | 房间朝向 | 下拉选择 | 东/南/西/北/东北/东南/西南/西北 |
| FR-01.7 | 常住人数 | 数字输入 | 单位：人 |

### FR-02 计算逻辑

根据输入参数，结合以下影响因子计算制冷量：
- 房间面积（基础冷负荷 120 W/m²）
- 城市气候系数（5 个气候分区）
- 房间朝向系数
- 楼层系数（顶层 1.15，非顶层 1.00）
- 窗户面积热负荷（200 W/m²）
- 人体散热（150 W/人）

### FR-03 输出结果

- 同时显示 **千瓦（kW）** 和 **匹（HP）**
- kW 保留一位小数（如 3.5 kW）
- 匹数保留一位小数（如 1.5 匹）
- 结果在页面下方以卡片形式展示

## 非功能需求

### NFR-01 界面设计
- Windows 11 设计语言：圆角、图标、字体、间距
- 主色调淡蓝色（#A2D2FF）
- 毛玻璃透明效果（CSS backdrop-filter）
- 无边框窗口（frameless）
- 字体：Segoe UI Variable

### NFR-02 运行环境
- Windows 11（兼容 Windows 10）
- 桌面应用形式（Electron 打包为 .exe）

### NFR-03 分发形式
- 单个 .exe 文件（portable），无需安装

## 验收标准

| 编号 | 验收条件 |
|------|----------|
| AC-01 | 所有 7 个输入参数可正常填写 |
| AC-02 | 计算结果与手工计算偏差 ≤ 0.1kW |
| AC-03 | 结果同时显示 kW 和匹数，各保留一位小数 |
| AC-04 | 窗口具有圆角、毛玻璃效果 |
| AC-05 | 界面以淡蓝色为主色调 |
| AC-06 | 双击 .exe 即可运行，无需安装 |
| AC-07 | 顶层房间制冷量 > 同条件非顶层房间 |
