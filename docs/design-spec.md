# UI/UX 设计规范

## 设计语言

遵循 Windows 11 Fluent Design System，核心特征：
- 圆角（Corner Radius）
- 层次感阴影
- 毛玻璃透明材质（Mica / Acrylic 效果）
- 淡蓝色主色调

## 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 | `#A2D2FF` | 淡蓝色，按钮、聚焦边框 |
| 主色调加深 | `#7BB8E8` | 按钮悬停状态 |
| 窗口背景 | `rgba(200, 220, 240, 0.55)` | 半透明淡蓝底 |
| 毛玻璃效果 | `backdrop-filter: blur(20px)` | 背景模糊 |
| 卡片背景 | `rgba(255, 255, 255, 0.75)` | 半透明白底 |
| 文字主色 | `#1A1A2E` | 深色文字 |
| 文字辅色 | `#5A5A7A` | 标签文字 |
| 输入框背景 | `rgba(255, 255, 255, 0.6)` | 半透明白底 |
| 输入框边框 | `rgba(160, 180, 210, 0.5)` | 淡蓝边框 |
| 聚焦边框 | `#A2D2FF` | 输入框聚焦发光 |
| 结果 kW 色 | `#FF6B6B` | 暖色强调 |
| 结果 匹色 | `#4ECDC4` | 冷色强调 |
| 阴影 | `rgba(0, 0, 0, 0.08)` | 轻微投影 |

## 尺寸规范

### 窗口
- 宽度：520px
- 高度：640px
- 圆角：12px

### 控件
- 输入框高度：36px
- 输入框圆角：8px
- 下拉框高度：36px
- 下拉框圆角：8px
- 按钮高度：44px
- 按钮圆角：8px
- 按钮宽度：200px

### 结果卡片
- 圆角：12px
- 内边距：20px 24px

### 间距
- 表单行间距：16px
- 标签与输入框间距：8px
- 表单区与按钮间距：24px
- 按钮与结果区间距：24px
- 页面左右内边距：32px

### 字体大小
- 窗口标题：16px，粗体 600
- 表单标签：13px，常规 400
- 输入文字：14px，常规 400
- 按钮文字：15px，粗体 600
- 结果标题：13px，常规 400
- 结果数值：28px，粗体 700
- 结果单位：14px，常规 400

## 字体

- 首选：`Segoe UI Variable`（Win11 系统字体）
- 回退：`Segoe UI`, `system-ui`, `sans-serif`

## 动效规范

| 元素 | 动效 | 时长 |
|------|------|------|
| 按钮悬停 | 背景色加深 + 轻微放大 | 200ms ease |
| 输入框聚焦 | 边框变色 + 外发光 | 200ms ease |
| 结果显示 | 淡入 + 上移 | 300ms ease |
| 窗口首次加载 | 整体淡入 | 400ms ease |

## Windows 11 特有处理

- 标题栏左侧预留 12px 圆角呼吸空间
- 窗口右下角不放调整手柄（固定尺寸）
- 最小化/关闭按钮采用 Win11 风格小圆点（snap layout 风格不强制实现）
- 所有可点击区域 cursor: pointer
- 文本不可选中（user-select: none）
