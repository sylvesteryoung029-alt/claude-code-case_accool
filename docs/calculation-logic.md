# 计算公式与参数说明

## 核心公式

```
总制冷量(W) = (面积 × 基础冷负荷 × 气候系数 × 朝向系数 × 楼层系数) + 窗户热负荷 + 人体散热
```

## 参数取值

### 1. 基础冷负荷

住宅建筑标准值：**120 W/m²**

（参考《民用建筑供暖通风与空气调节设计规范》GB 50736 中住宅单位面积冷负荷指标）

### 2. 气候系数

基于中国《民用建筑热工设计规范》(GB 50176) 气候分区：

| 气候分区 | 代表城市 | 系数 | 夏季特征 |
|----------|----------|------|----------|
| 严寒地区 | 哈尔滨、长春、乌鲁木齐、呼和浩特、拉萨 | 0.70 | 凉爽短暂 |
| 寒冷地区 | 北京、天津、沈阳、西安、石家庄、太原、济南、郑州、兰州、西宁、银川 | 0.85 | 较热，持续时间短 |
| 夏热冬冷 | 上海、南京、武汉、重庆、成都、杭州、合肥、南昌、长沙 | 1.00 | 炎热，基准值 |
| 夏热冬暖 | 广州、深圳、福州、南宁、海口、厦门 | 1.15 | 漫长炎热 |
| 温和地区 | 昆明、贵阳 | 0.80 | 四季如春 |

### 3. 朝向系数

| 朝向 | 系数 | 说明 |
|------|------|------|
| 北 | 0.90 | 最少日照 |
| 东北 | 0.95 | 仅早晨有日照 |
| 西北 | 0.95 | 仅傍晚有日照 |
| 东 | 1.05 | 上午日照 |
| 西 | 1.05 | 下午日照（夕晒） |
| 东南 | 1.00 | 基准 |
| 西南 | 1.00 | 基准 |
| 南 | 1.10 | 全天日照最多 |

### 4. 楼层系数

| 条件 | 系数 | 说明 |
|------|------|------|
| 非顶层 | 1.00 | 上下有楼层隔热 |
| 顶层 | 1.15 | 屋顶受太阳直射，热负荷增加约 15% |

### 5. 窗户面积热负荷

```
窗户热负荷(W) = 窗户面积(m²) × 200 W/m²
```

窗户的隔热性能远低于墙体，单位面积得热量约为墙体的 1.5-2 倍，取 200 W/m²。

### 6. 人体散热

```
人体散热(W) = 人数 × 150 W/人
```

成年人在静坐状态下散热量约 100-150W，取中间偏上值 150W/人。

## 单位换算

| 单位 | 换算关系 |
|------|----------|
| 1 匹 | ≈ 2500 W = 2.5 kW |
| 1 kW | = 1000 W |

> 注：空调行业中 1 匹（HP）的实际制冷量因品牌而异（2200W-2800W），本工具取工程常用值 2500W。

## 计算步骤

```
步骤 1: 基础负荷 = 房间面积 × 120
步骤 2: 查表获取气候系数（根据所选城市）
步骤 3: 查表获取朝向系数（根据所选朝向）
步骤 4: 判断楼层系数（顶层 1.15，否则 1.00）
步骤 5: 墙体负荷 = 基础负荷 × 气候系数 × 朝向系数 × 楼层系数
步骤 6: 窗户负荷 = 窗户面积 × 200
步骤 7: 人体负荷 = 人数 × 150
步骤 8: 总制冷量(W) = 墙体负荷 + 窗户负荷 + 人体负荷
步骤 9: kW = 总制冷量 / 1000（保留一位小数）
步骤 10: 匹 = 总制冷量 / 2500（保留一位小数）
```

## 示例计算

**输入：**
- 城市：广州（夏热冬暖，系数 1.15）
- 面积：20 m²
- 窗户：3 m²
- 楼层：5 层，非顶层（系数 1.00）
- 朝向：南（系数 1.10）
- 人数：2 人

**计算过程：**

```
基础负荷    = 20 × 120 = 2400 W
墙体负荷    = 2400 × 1.15 × 1.10 × 1.00 = 3036 W
窗户负荷    = 3 × 200 = 600 W
人体负荷    = 2 × 150 = 300 W
总制冷量    = 3036 + 600 + 300 = 3936 W
kW          = 3936 / 1000 = 3.9 kW
匹          = 3936 / 2500 = 1.6 匹
```

## 边界处理

| 情况 | 处理方式 |
|------|----------|
| 面积 ≤ 0 | 弹出提示"请输入有效的房间面积" |
| 窗户面积 ≤ 0 | 允许为 0（无窗户房间） |
| 窗户面积 > 房间面积 | 弹出提示"窗户面积不能大于房间面积" |
| 人数 < 0 | 不允许（input min=1） |
| 楼层数 < 1 | 不允许（input min=1） |
| 未选城市/朝向 | 弹出提示"请完善所有选项后再计算" |
